# دليل تثبيت وتوزيع تطبيق الـ PWA 🚀 | PWA Installation & Deployment Guide

تم إعداد هذا المشروع وترقيته بالكامل ليكون **تطبيق ويب تقدمي (PWA - Progressive Web App)** مستقل تماماً ومُحسّن لمختلف منصات التشغيل (Android، وiOS، وDesktop) دون الحاجة لملفات متجر Google Play معقدة أو عمليات بناء ضخمة.

عند تنزيل مشروعك أو نشره، ستكون الملفات مقتصرة بالكامل على الـ PWA ومُدمج معها تخزين مؤقت متكامل للعمل دون إنترنت (Offline-First).

---

## ✨ ميزات تطبيق الـ PWA الحالي
1. **تثبيت مستقل (Standalone UI)**: يتم تثبيت التطبيق على الهواتف والكمبيوتر كشاشة كاملة بدون أشرطة متصفح، مما يعطي تجربة التطبيقات الأصلية (Native Feel).
2. **العمل دون إنترنت (Offline Access)**: يقوم ملف الـ `sw.js` تلقائياً بحفظ حزم الويب، الأنماط (CSS)، لغة البرمجة (JS) والأيقونات محلياً بعد أول تشغيل للتطبيق.
3. **أيقونة مخصصة احترافية**: تم دمج أيقونة مخصصة واحترافية للمؤشرات المالية (`icon-512.png`) لتظهر على الشاشة الرئيسية للهاتف بمجرد التثبيت.
4. **خفيف الحجم وسريع الاستجابة**: يستهلك بضع كيلوبايتات فقط ولا يحتاج لتحديثات دورية متعبة من المتاجر.

---

## 📱 كيفية تثبيت التطبيق على مختلف الأجهزة

### 🤖 1. على هواتف أندرويد (Chrome / Edge / Samsung Internet)
* افتح متصفح **Chrome** واذهب لرابط التطبيق الخاص بك.
* ستظهر لك لافتة أسفل الشاشة أو في القائمة العلوية باسم **"إضافة إلى الشاشة الرئيسية" (Add to Home Screen)** أو **"تثبيت التطبيق" (Install App)**.
* اضغط عليها وسيتم تحميل التطبيق ووضع أيقونة الشارت الاحترافي مباشرة بجوار تطبيقاتك الأخرى!

### 🍏 2. على هواتف آيفون وأجهزة آيباد (Safari)
* افتح متصفح **Safari** على جهازك واذهب للرابط.
* اضغط على زر **المشاركة (Share Button)** (المربع الذي يحتوي على سهم يشير للأعلى).
* مرر لأسفل وحدد **"إضافة إلى الشاشة الرئيسية" (Add to Home Screen)**.
* اضغط على **إضافة (Add)** في الجزء العلوي الأيمن، وستجد التطبيق بشاشته الكاملة الفاخرة يعمل فوراً على هاتفك.

### 💻 3. على أجهزة الكمبيوتر (Windows / Mac / Linux)
* افتح المتصفح (Chrome أو Edge).
* ستجد في شريط العنوان (URL bar) أيقونة صغيرة على شكل كمبيوتر مع شاشة أو علامة **"+"**.
* اضغط عليها واصنع **تثبيت (Install)** ليتحول التطبيق لبرنامج مستقل بالكامل على حاسوبك الشخصي.

## 🌐 كيفية نشر واستضافة التطبيق على الإنترنت

لجعل التطبيق قابلاً للتثبيت، تحتاج فقط لرفعه على أي استضافة تدعم بروتوكول الأمان **HTTPS** (مطلوب إلزامياً لتفعيل الـ Service Worker). أفضل الخيارات المجانية والسريعة:

1. **Vercel** ⚡ (الأسهل والأسرع لـ React Vite)
2. **Netlify** 🌐
3. **GitHub Pages** 🐙
4. **Cloud Run / Docker** ☁️ (باستخدام السيرفر الافتراضي المرفق مع المشروع `server.ts`)

---

## 🏗️ خطوات البناء اليدوي للمطورين
لتشغيل أو بناء ملفات الـ PWA محلياً على جهازك:

```bash
# 1. تثبيت التبعيات
npm install

# 2. تشغيل الوضع التطويري المباشر
npm run dev

# 3. بناء النسخة الإنتاجية (الملفات ستكون داخل مجلد dist جاهزة للنشر!)
npm run build

# 4. تشغيل خادم الإنتاج الفعلي برمجياً
npm run start
```
